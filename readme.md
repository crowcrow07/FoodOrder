1. useState, useEffect : 사용하는데에 문제 없었다.
2. useRef, forwardRef : forwardRef라는걸 모르고있어서 이번에 알게되는 계기가 되었다.
3. Portal : Portal을 통해 모달을 안정적으로 만들었다.
4. Context API : redux 쓰기 애매한 상황에서 전역값을 관리해야할때 새로운 선택지를 배우게 되었다.
